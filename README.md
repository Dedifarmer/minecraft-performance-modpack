normal:
  -large
  -more fps
  -more compatibility
  -needs more ram
  -stable
  
vulkan:
  -small
  -lower fps
  -needs less ram
  -running on vulkan instead of opengl
  -only runs on windows10/11
  -needs a compatible gpu
  -somtimes unstable
